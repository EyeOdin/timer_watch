# Purpose

Timer Watch is a Krita plugin and is a Time Management Tool.

# Features

* Clock
* Stopwatch
* Progress Bar
* Pause
* Alarm
* Read the ![wiki](https://github.com/EyeOdin/timer_watch/wiki) for more information
