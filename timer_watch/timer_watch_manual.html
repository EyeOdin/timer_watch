Timer Watch

Mode

Clock : Display operating system clock, includes Stopwatch progress bar.
Stopwatch : Display Stopwatch.
Info : Display document information.

Stopwatch

Start/Pause : Start will start the Timer and it does not stop until the user cancels. Pause will pause the timer progress to take a break.
Reset : Stop the timer and reset.
Alarm : Stop and Resets the timer upon reaching the time limit. Shows a warnning message upon reaching the time goal.
Limit : Time limit to be spent for the stopwatch progress bar.
Progress Bar: Progress status expressed in a percentage of the amount of time limit set, if 00:00:00 is set it will become grey.

Information

Title : Title of the current active document.
Initial Creator : Name of the author of the document.
Creator Name : First and last name of the last author editing.
Nickname : Nickname of the last author editing.
Contact Type : Contact address of the last author editing.
Creation Date : Creation date time stamp.
Date Delta : Amount of time that has passed "since creation" of the document.
Editing Time : Active editing time in cycles.
Active Time : Active editing time in formated time.
Editing Cycles : Time passing in cycles.
