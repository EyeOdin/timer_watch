DOCKER

Mode : Press neutral LMB to change modes. Press RMB to enter the context menu and select the mode.
Clock : Display operating system clock, includes Stopwatch progress bar.
Stopwatch : Display Stopwatch.


STOPWATCH

Progress Bar: Progress status expressed in a percentage of the amount of time limit set, if 00:00:00 is set it will become grey.
Start/Stop : Start will start the Stopwatch counter. Stop will pause the Stopwatch counter progress, useful for breaks.
Reset : Stops and Resets the Stopwatch counter.
Alarm : Activate a warnning message and sound upon reaching the time goal.
Limit : Setup the progress bar time range limit.
Settings : Display copyright.
