Timer Watch

Operators

Time Display : Time Progress Display.
Start : Starts the Timer and it does not stop until the user cancels.
Pause : Pause the timer progress to take a break.
Stop : Stop the timer and reset.
Alarm : Stop and Resets the timer upon reaching the time limit, while showing a warnning message.
Limit : Places a amount of time to be spent for the progress bar.
Progress : Progress status expressed in a percentage of the amount of time limit set, if 00:00 is set it will return a error message.

Information

Title : "Title" of the current active document.
Creator : "Nickname" and "Name" (first and last) of the Creator.
Author : Current active "Author" and their "Contact Type".
Date : "Creation Date" and "Date Delta" since creation.
Edit : "Editing Time" spent on the active document and this value is only updated upon saving the file.
Cycle : "Editing Cycles" is the amount of time the document was and is open.
"Editing Time" and "Editing Cycles" values can be Reset by : File > Document Information > General > Reset.
